github cli

commit and PR via cli

git status
git commit -m "files updated"
git push origin main

gh pr create --tile "files updated" --body "update gitignore or general Note"

--approve current branch
gh pr review --approve

gh pr review 123 --approve --> pr by number

gh pr re view https://github.com --approve --> pr via url

gh pr 123 --approve -b "code looks great"  --> PR with approval msg
